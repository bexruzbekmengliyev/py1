Pythonda 1-dars

Python tili bu xozirgi kunda eng ommabop tillardandir. U Guido van Rossum tomonidan yaratilgan va 1991-yilda chiqarilgan.

  • veb-ishlab chiqish (server tomonida),
  • dasturiy ta'minotni ishlab chiqish,
  • matematika,
  • tizim skriptlari.

Nima uchun Python?

  • Python turli platformalarda ishlaydi (Windows, Mac, Linux, Raspberry Pi va boshqalar).

    Python ingliz tiliga o'xshash oddiy sintaksisga ega.

  • Python dasturchilarga boshqa dasturlash tillariga qaraganda kamroq qatorli dasturlar yozish imkonini beruvchi sintaksisga ega. Python interpretator tizimida ishlaydi, ya'ni kod yozilgan zahotiyoq bajarilishi mumkin. Bu shuni anglatadiki, prototiplash juda tez bo'lishi mumkin. Pythonni protsessual, obyektga yo'naltirilgan yoki funktsional usulda ko'rib chiqish mumkin.
